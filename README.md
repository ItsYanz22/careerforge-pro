# 🚀 CareerForge Pro

### ATS-Proof AI Resume Generator & Job Matcher

CareerForge Pro is an AI-powered Resume Architect platform that helps users create ATS-optimized resumes, analyze Job Descriptions, improve resume content using AI, and generate professional exports in PDF/DOCX formats.

Built with modern frontend architecture, AI-assisted resume optimization, and SaaS-ready scalability.

# ⚙️ Getting Started

## 1️⃣ Clone Repository

```bash
git clone <your-repository-url>
cd careerforge-pro
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Start Development Server

```bash
npm run dev
```

Application runs at:

```bash
http://localhost:5173
```
